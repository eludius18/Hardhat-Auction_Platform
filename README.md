# Create an Auction Project

## Install Dependencies

```shell
npm install
```

## Create .env and add the info

```shell
RPC_URL=''
PRIVATE_KEY=''
```

## Install .env

```shell
npm install dotenv
```

## Compile

```shell
npx hardhat compile
```

## Deploy contract on Ropsten

```shell
npx hardhat run --network ropsten scripts/deploy.js  
```
